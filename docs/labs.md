# Practice Labs

The Practice Labs are notebooks you can execute to understand what the CWL CommandLineTools and Workflows do in the context of this module.

There are four sets of notebooks, each set with one or more notebooks:

1 - **CWL CommandLineTools**

This first set of notebooks targets the Producer Application Package individual CommandLineTools. By running these notebooks, you'll generate the intermediary outputs of the Producer Application Package

The notebooks included are:

- [1. Discovery](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/1-CWL-CommandLineTools/1-Discovery.ipynb)
- [2. Convert-search](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/1-CWL-CommandLineTools/2-Convert-search.ipynb)
- [3. Water bodies detection](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/1-CWL-CommandLineTools/3-Water-bodies-detection.ipynb)
- [4. STAC Catalog](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/1-CWL-CommandLineTools/4-STAC-Catalog.ipynb)
- [5. STAC Zarr](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/1-CWL-CommandLineTools/5-STAC-Zarr.ipynb)

2 - **Producer CWL Workflow**

Run the producer CWL Workflow that chains the CommandLineTools and sub-workflow:

- [1. Producer](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/2-Producer-CWL-Workflow/1-Producer.ipynb)

3 - **Exploitation**

Read and inspect the results generated by the **Producer CWL Workflow**. 

- [1. Exploitation](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/3-Exploitation/1-Exploitation.ipynb)

4 - **Consumer CWL Workflow**

Run the consumer CWL Workflow that reads the outputs generated by the **Producer CWL Workflow**:

- [1. Consumer](https://github.com/eoap/zarr-cloud-native-format/blob/main/practice-labs/4-Consumer-CWL-Workflow/1-Consumer.ipynb)
